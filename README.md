## Developer Notes
```sh
# load dependencies
npm i
# create a local executable
npm link
# run it!
npx aoc --day dec01 -t arrays -t integer --part 2
# run it on a previous year
npx aoc --year 2021 --day dec01 -t lines -t integer -v
# start a new AoC day (copies templates/):
npm run newDay dec02
```
